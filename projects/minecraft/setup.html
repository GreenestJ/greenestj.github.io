General Info

Download one of the main launchers I use MultiMC or Curseforge (I've heard ATLauncher is a good alternative but I don't use it). I recommend MultiMC but its less user friendly than Curseforge also if you have a Microsoft account you can't use MultiMC currently.

My mods will have 3 versions to download which is which program I exported from. The MultiMC's exports as far as I know is not compatible with curseforge. The CurseForge versions are fully compatible with MultiMC as far as I know. The last version is just a straight compressed 7z with the mods and files as an extra backup but this cannot be imported into launchers this is meant to be extracted (this is pretty similar how MultiMC handles exports but its to be 100% safe).

For other mod launchers look up how to use them as I won't cover them. Base minecraft launcher is very not recommended to be used.

File Size Discrepancy and Possible Bugs: MultiMC exports the files as a zip while CurseForge exports the file as a big list of the mods so its much smaller in file size. If the files are deleted from the CurseForges site it can no longer fetch them and breaks mods so MultiMC is more reliable. Compressed is just the actual compressed files by me no use of launchers.

Importing

MultiMC

1) Download whatever modpack you're playing
DO NOT UNZIP THE MOD UNLESS ITS SOMETHING OTHER THAN .ZIP

2) Launch MultiMC

3) In the top left click on add instance

4) Choose import from zip

5) Click browse and find the zip in your files

5.1) I recommend making a group for the pack by typing a group name in the group field but this is not necessary if you don't plan to use MultiMC a lot.

6) Click Ok and the mod pack will start importing. While importing the program will stop responding it is working fine it just works like that for some reason so wait for it to respond again.

7) Mod Packs almost ready to go all thats left is dedicating ram which is below

CurseForge

1) Download whatever modpack you're playing
DO NOT UNZIP THE MOD UNLESS ITS SOMETHING OTHER THAN .ZIP

2) Click create custom profile

3) At the top below Create Profile it says or Import a previously created profile click on import

4) Select the mod pack

5) It should auto install (I don't use Curse anymore so I'm not sure) and be ready except ram dedication which is below

Extra

If you download the compressed version of my mods the set up is a little different than the other versions.

1) Create an instance in whatever launcher you're using.

2) Get to the instances folder in windows file explorer.

3) Extract the compressed file.

4) Transfer the new uncompressed files into the instances folder.

How to Dedicate Ram

WARNING: Dedicating too much ram can cause its own issues as explained in this blog post by vazkii a prominent modder. The best method to figure out how much ram is needed is to experiment by dedicating a base line and increasing it by 512 MB/0.5 GB if it crashes or is laggy.

Keep in mind a CPU can also cause issue due to how minecraft was designed that no amount of ram can fix (same reason why servers can't hold 100+ people)

MultiMC

1) Click on the mod pack to have it selected.

2) On the right side click on Edit Instance

3) Click on Settings

4) Check the memory box to allow custom values

5) Set the minimum and maximum ram to what you want (1 Gigabyte is 1024 Megabytes)

6) It's ready you can just close the setting windows it auto saves and launch the pack

CurseForge

1) Click on the pack and be in its menu

2) Click on the 3 dots next to play then profile options

3) Uncheck use system memory and adjust the bar to whatever you want
(The minimum and maximum ram is set the same | 1 GB is 1024 MB)

4) Click Done and you're ready to use the pack

Errors
